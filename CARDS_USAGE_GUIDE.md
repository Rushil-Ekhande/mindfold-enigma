# Quick Guide: Using the Morphing Card Stack

## What's New?

The admin therapists page now has an interactive card display with multiple viewing modes!

---

## How to Use

### Step 1: Access the Page
Navigate to `/admin/therapists` in your admin dashboard.

### Step 2: Choose Your View Mode

**Cards Mode** (New! üéâ)
- Click the "Cards" button in the top right
- See therapists as interactive cards
- Three layout options available

**List Mode** (Original)
- Click the "List" button in the top right
- Traditional vertical list view
- All details visible at once

---

## Cards Mode - Layout Options

### 1. Stack Layout (üìö)
- Cards stacked on top of each other
- **Swipe left/right** to navigate
- Click pagination dots to jump to specific card
- Best for: Focused review of one therapist at a time

### 2. Grid Layout (‚äû)
- 2-column grid of cards
- See multiple therapists at once
- Click any card to view details
- Best for: Quick overview and comparison

### 3. List Layout (‚ò∞)
- Vertical list of cards
- Similar to grid but single column
- Scroll to see all therapists
- Best for: Mobile viewing

---

## Interacting with Cards

### Click a Card
1. Click any card in the display
2. Full details appear below
3. View qualifications, documents, and actions
4. Click another card to switch

### Swipe Cards (Stack Mode Only)
1. Click and drag a card left or right
2. Release to navigate to next/previous
3. Or use pagination dots below

### Approve/Reject
1. Click a card to view details
2. Scroll to action buttons
3. Click "Approve" or "Reject"
4. For reject: Enter reason in dialog

---

## Filtering

Use the filter tabs at the top:
- **Pending**: Only pending applications (default)
- **Approved**: Only approved therapists
- **Rejected**: Only rejected applications
- **All**: Show everyone

Filters work in both Cards and List modes!

## Common Workflows

### Reviewing New Applications
1. Filter: "Pending"
2. View: "Cards" ‚Üí Stack layout
3. Swipe through each application
4. Click card to view documents
5. Approve or reject with reason

### Checking Approved Therapists
1. Filter: "Approved"
2. View: "Cards" ‚Üí Grid layout
3. Quick overview of all approved
4. Click any to view details

### Finding Specific Therapist
1. Filter: "All"
2. View: "List"
3. Scroll through list
4. Or use browser search (Ctrl+F)

## Troubleshooting

### Cards Not Showing?
- Check if filter has results
- Try switching to "All" filter
- Refresh the page

### Swipe Not Working?
- Make sure you're in Stack layout
- Click and drag (don't just click)
- Try on the top card only

### Details Not Appearing?
- Make sure you clicked a card
- Scroll down to see details section
- Try clicking the card again
